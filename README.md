# VAWT Power Generation — Minor Project

A Vertical Axis Wind Turbine (VAWT) installed between highways to harness 
wind energy generated by passing vehicles. This Arduino-based system 
includes real-time voltage/current sensing and an MPPT (Maximum Power 
Point Tracking) algorithm with a boost converter for stable output.

---

## How It Works

Wind from passing highway vehicles spins the VAWT. The turbine generates 
variable DC voltage which is fed into a boost converter. An Arduino UNO 
continuously monitors the turbine voltage, current, and boost output, 
then adjusts the PWM duty cycle using an MPPT algorithm to extract 
maximum power at all times.

---

## Components Used

| Component | Purpose |
|---|---|
| Arduino UNO | Main controller |
| ACS712 Current Sensor | Measures turbine output current |
| Voltage Divider (10kΩ / 4.7kΩ) | Reads turbine voltage |
| Voltage Divider (47kΩ / 10kΩ) | Reads boost converter output |
| Boost Converter | Steps up turbine voltage |
| MOSFET + PWM (Pin 9) | Controls boost converter duty cycle |

---

## Pin Connections

| Arduino Pin | Connected To |
|---|---|
| A0 | ACS712 current sensor output |
| A1 | Turbine voltage divider |
| A2 | Boost converter output voltage divider |
| D9 (PWM) | Boost converter MOSFET gate |

---

## MPPT Algorithm

The code uses a **Perturb & Observe (P&O)** method:
- If output power increases → continue in same duty direction
- If output power decreases → reverse duty direction
- Duty cycle constrained between **10 and 230** (out of 255)

### Voltage protection limits
- Above **14.2V** → immediately reduce duty (overvoltage protection)
- Between **12.5V – 14.2V** → fine tuning mode
- Below **12.5V** → full MPPT mode active

## Sensor Calibration

The ACS712 offset is auto-calibrated at startup by averaging 500 ADC 
readings — no manual zero adjustment needed.
